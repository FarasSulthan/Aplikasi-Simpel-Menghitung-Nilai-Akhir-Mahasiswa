# Aplikasi Simpel Menghitung Nilai Akhir Mahasiswa

Deskripsi singkat
-----------------
Program kecil ini dibuat buat bantu mahasiswa ngitung nilai akhir berdasarkan komponen penilaian (mis. tugas, kuis, UTS, UAS). Tujuannya buat latihan logika perhitungan dan penerapan input/output di Python.

Fitur
-----
- Input nilai per komponen.
- Hitung nilai akhir berdasarkan bobot di skrip.
- Tampilkan nilai akhir dan grade/predikat sederhana.

Cara pakai (singkat)
-------------------
1. Buka terminal di folder proyek.
2. Jalankan:

```bash
python Aplikasi_Menghitung_Nilai_Akhir_Mahasiswa.py
```

3. Masukkan nilai saat diminta, hasil akan ditampilkan di layar.

Contoh
------
Contoh input:
- Tugas: 80
- Kuis: 75
- UTS: 70
- UAS: 85

Contoh output (bergantung rumus di skrip):

```
Nilai akhir: 78.5 — Grade: B
```

Catatan penting
---------------
- Bobot dan skema grading diatur langsung di `Aplikasi_Menghitung_Nilai_Akhir_Mahasiswa.py`.

Struktur proyek
---------------
- `Aplikasi_Menghitung_Nilai_Akhir_Mahasiswa.py` — skrip utama
